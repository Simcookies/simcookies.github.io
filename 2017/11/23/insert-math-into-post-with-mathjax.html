Insert Math into post with MathJax

Nov 23, 2017    blog    jekyll  latex 

Sometimes we may need to insert math equation into posts. MathJax can help us to do this. We can write LaTeX-like equations in posts with MathJax. And it’s also easy to config into Jekyll.

Add script tag into HTML head.

Add the script below into every page (or add it into template page “default.html”):

<script src="https://cdn.mathjax.org/mathjax/latest/MathJax.js?config=TeX-AMS-MML_HTMLorMML"></script>

Use redcarpet or karmdown as Markdown compiler1

Set it in _config.yml file:

markdwon: kramdown # or redcarpet

Use LaTeX-like code

MathJax use \\( \\) for inline math and \\[ \\] or $$ $$ for displayed math. For instance:

$$ E = mc^2 $$

You can get:

\\( \alpha\times\beta\times\gamma \\) can get \(\alpha\times\beta\times\gamma\)

  1. Because some strange things will happen when you use default Markdown compiler. ↩
